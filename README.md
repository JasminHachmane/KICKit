# KICKit UI Proof of Concept v1

## Overview

This project is the first version of the KICKit user interface.

It is a proof of concept for an interactive football experience for children in hospitals.  
This version is made to show how the interface, screens, and basic flow could work.

This is not the final product.

## Current status

This is version 1 of the UI.

It is an early prototype used to:
- test the flow
- show the concept
- collect feedback
- improve the experience in later versions

Some parts are still visual only and not fully connected to real hardware or final game logic.

## What this version includes

The prototype currently includes:

- a fullscreen start screen
- a hand detection screen placeholder
- mascot selection
- player profile setup
- game mode selection
- a playable game screen
- a result screen
- custom background images and mascot images
- basic game feedback and sound support

## What this version does not include yet

This version is not finished yet.

It does not include:
- real hand detection
- real camera tracking
- final print flow
- final animations
- full technical integration
- final gameplay balancing
- final hospital screen testing

## Project goal

The goal of this version is to build and test the first user interface for the KICKit concept.

It is meant to answer questions like:
- Does the flow make sense?
- Do the screens feel clear and friendly?
- Does the concept work as a first interactive experience?
- What should be improved in the next version?

## User flow

The current user flow is:

1. Start screen  
2. Hand detection screen  
3. Mascot selection  
4. Profile setup  
5. Game mode selection  
6. Game screen  
7. Result screen  

## How to run
Put all files in one project folder
Make sure the assets folder exists
Add the required images and audio files
Open startscherm.html in the browser

## Notes

This project is a first UI prototype only.

It should be seen as:

a first interface version
a proof of concept
a base for testing and improvement

It is not finished and will need more development before it can be used as a final product.

KICKit UI Proof of Concept v1 is the first interactive interface prototype for the KICKit experience.
